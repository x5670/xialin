# v2ray
最好用的 V2Ray 一键安装脚本 &amp; 管理脚本


安装 V2Ray
输入下面命令回车，你可以复制过去，然后在 Xshell 界面按 Shift + Insert 即可粘贴，不能按 Ctrl + V 的。。


bash <(curl -s -L https://233yes.com/v2ray.sh)

V2Ray 管理面板
现在可以尝试一下输入 v2ray 回车，即可管理 V2Ray
